# 🎬 Next.js Movie App

## 🎯 What is this?
A simple movie app that shows movie information. Built with Next.js 12 and TMDB API.

## ✨ What can it do?
- 🎯 Show movie list
- 🔍 Show movie details
- 📱 Works well on mobile
- 🎨 Nice and clean design

## 🛠 Built with
- ⚛️ Next.js 12
- ⚡ React 17
- 🌐 TMDB API

## 🚀 How to start

### You need
- Node.js
- npm or yarn
- TMDB API key

### Setup steps
1. Clone this project
```bash
git clone [repository-url]
```

2. Install packages
```bash
npm install
# or
yarn install
```

3. Make `.env` file and add your TMDB API key
```
API_KEY=your_tmdb_api_key_here
```

4. Start the app
```bash
npm run dev
# or
yarn dev
```
